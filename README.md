# P4-80x40-LED-Matrix-with-Arduino-Uno-R4-WiFi

A beginner-friendly project for driving a P4 80x40 1/10 scan LED matrix using the Arduino Uno R4 WiFi.

## Features
- Pixel drawing
- Basic graphics
- Digital clock
- Timer display

## Hardware
- Arduino Uno R4 WiFi
- P4 80x40 LED Matrix Panel
- HUB75 connector
